Política de Privacidade

1. Informações que coletamos

O PizzaKing coleta apenas as informações estritamente necessárias para o funcionamento do app:

  • Nome e e-mail do Google — obtidos via login com Google, usados apenas para identificar a conta e sincronizar o histórico de eventos.
  • Dados de eventos — nomes de participantes, quantidades de fatias e aperitivos registados durante o uso do app.

Não coletamos localização, contatos, câmera, microfone ou qualquer outro dado pessoal.

2. Como usamos as informações

  • Sincronizar o histórico de eventos entre dispositivos quando o utilizador está logado.
  • Identificar a conta para salvar e recuperar dados no servidor.

Não usamos os dados para publicidade, análise de comportamento ou qualquer finalidade além das listadas acima.

3. Armazenamento e segurança

Os dados são armazenados de forma segura no Firebase Firestore (Google Cloud), protegidos por regras de acesso que garantem que cada utilizador só acede aos seus próprios dados.

Uma cópia local é mantida no dispositivo via AsyncStorage para funcionamento offline.

4. Compartilhamento de dados

Não vendemos, alugamos nem partilhamos os seus dados com terceiros. Os dados são processados exclusivamente pelos serviços Firebase (Google) conforme a Política de Privacidade do Google.

5. Exclusão de dados

O utilizador pode eliminar todos os seus dados a qualquer momento. Para solicitar a exclusão completa da conta e dos dados associados, envie um e-mail para o endereço indicado na secção de contato abaixo.

6. Crianças

O PizzaKing não é direcionado a menores de 13 anos e não recolhe intencionalmente dados de crianças.

7. Alterações a esta política

Caso esta política seja alterada, a data de atualização no final desta página será modificada. O uso continuado do app após alterações implica a aceitação da nova política.

Termos de Uso

1. Aceitação dos termos

Ao instalar ou utilizar o PizzaKing, o utilizador concorda com estes Termos de Uso. Se não concordar, deve desinstalar o aplicativo.

2. Descrição do serviço

O PizzaKing é um aplicativo gratuito para registo e acompanhamento do consumo de fatias de pizza e aperitivos em eventos sociais. O app permite criar rankings de participantes e salvar o histórico de eventos.

3. Conta de utilizador

O login com Google é opcional. Sem login, os dados são armazenados localmente no dispositivo. Com login, os dados são sincronizados com a nuvem. O utilizador é responsável pela segurança da sua conta Google.

4. Uso adequado

O utilizador compromete-se a não usar o app para fins ilícitos, não tentar aceder a dados de outros utilizadores e não interferir no funcionamento do serviço.

5. Disponibilidade

O app é fornecido "tal como está", sem garantias de disponibilidade contínua. Podemos interromper ou modificar o serviço a qualquer momento sem aviso prévio.

6. Limitação de responsabilidade

O PizzaKing não se responsabiliza por perdas de dados, danos diretos ou indiretos decorrentes do uso do aplicativo.

7. Alterações aos termos

Estes termos podem ser atualizados a qualquer momento. O uso continuado do app implica a aceitação dos termos vigentes.
